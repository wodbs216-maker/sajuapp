# 사주노트 v6.1

비개발자용 설명:
- `index.html`: 사이트의 입구/화면 뼈대
- `css/style.css`: 색상, 글자 크기, 카드 모양
- `data/character-data.js`: 카톡·연애·상황별 문장 사전
- `js/storage.js`: 최근 본 사람 저장
- `js/app.js`: 사주 계산, 화면 전환, 궁합 등 앱 동작
- `assets/`: 이미지 파일

GitHub에는 이 폴더 안의 파일과 폴더를 그대로 올리세요. ZIP 파일 자체를 저장소 루트에 올리는 방식은 권장하지 않습니다.

Cloudflare Pages 설정:
- Framework preset: None
- Build command: 비움
- Build output directory: `/` 또는 저장소 루트
