사주 캐릭터 연구소 v6 사용 안내

1. GitHub에 올릴 때
- saju-character-v6-github.zip 압축을 먼저 풉니다.
- 압축을 푼 폴더 안의 index.html, css, js, data, assets 폴더를 GitHub 저장소에 그대로 올립니다.
- ZIP 파일 하나만 저장소에 올리는 방식은 사용하지 않습니다.

2. Cloudflare Pages
- GitHub 저장소를 연결합니다.
- Framework preset: None
- Build command: 비워둠
- Build output directory: 저장소 루트(/)

3. 파일 역할
- index.html: 화면의 입구
- css/style.css: 글자 크기, 색상, 카드 모양
- data/character-data.js: 캐릭터·상황별 문장
- js/storage.js: 최근 본 사람 저장
- js/app.js: 계산·궁합·화면 전환
- assets/: 이미지

4. 저장 기능
- 계산한 사람은 최근 본 사람 목록에 자동 저장됩니다.
- 최대 8명까지 저장됩니다.
- 같은 사람을 다시 보면 최신 순서로 올라옵니다.
- 브라우저에만 저장되므로 다른 휴대폰과 자동 동기화되지는 않습니다.

5. 이번 v6 수정
- 반복되던 ‘자기 기준과 자존심…’ 문장 제거
- 매 카드마다 나오던 설명 문장 삭제
- 질문 글자를 크게, 답변 글자를 읽기 좋게 조정
- 상황별 카드에 아이콘과 부드러운 색상 적용
- 최근 본 사람 저장 기능 복원 및 자동 저장
- HTML/CSS/JS/콘텐츠 데이터를 분리해 유지보수 가능하게 변경
